1. Install dependencies for both the backend and frontend. From root:
	backend:
		cd backend
		pip install -r requirements.txt
	frontend:
		cd frontend
		npm install

2. Run the backend:
	cd backend
	python manage.py runserver

3. Run the frontend:
	cd frontend
	npm run dev

4. Go to 'http://localhost:5173/chat' in your browser of choice to access the chatbot.

Standard tooling and specified dependencies were used to correctly configure the frontend and backend environments.
