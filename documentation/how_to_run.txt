1. install the required libraries (a virtual environment is recommended).
	`pip install -r requirements.txt`
		OR
	`pip3 install -r requirements.txt`

2. run the server.
	`python capstone/manage.py runserver`
		OR
	`python3 capstone/manage.py runserver`

3. go to the following URL in a web browser:
	http://127.0.0.1:8000
